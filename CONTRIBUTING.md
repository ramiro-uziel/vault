# Contributing to { vault }

Thanks for your interest :)

## Project Status

Due to time restrictions, I can't review or merge large feature PRs.
The app is under solo development for now.

## How to Contribute

### Reporting Bugs

Before reporting:

- Check existing issues to avoid duplicates
- Use the latest version to see if the issue has been fixed

When reporting, please include:

- Release version
- Expected vs actual behavior
- Relevant logs or screenshots
- Steps to reproduce

### Suggesting Features

Please open an issue to share your idea.

### Questions & Support

For questions or troubleshooting, open a new discussion thread.

## Pull Requests

These are welcome:

- Bug fixes
- Typos and documentation improvements
- Small, focused changes

Avoid for now:

- Large feature additions
- Major refactors

Please open an issue if you are not sure about your PR.
